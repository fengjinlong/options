# DCF 估值工具

## 概述

DCF（Discounted Cash Flow，折现现金流）估值工具，专为高成长科技股设计，适用于英伟达、特斯拉、拼多多等公司的估值分析。

## 页面布局

页面采用左右分栏布局：
- **左侧**：参数输入区域
- **右侧**：估值结果展示

## 参数说明

### 基础参数

| 参数 | 说明 | 单位 | 示例 |
|------|------|------|------|
| 当前价格 (P) | 股票当前市场价格 | 元 | 176 |
| 自由现金流 (FCF) | 企业产生的自由现金流 | 元/亿 | 96600 亿 |
| 折现率 (r) | 投资者要求的回报率 | % | 17% |
| 增长率 (g) | 1-5 年增长率 | % | 38.8% |
| 增长率 (g₂) | 6-10 年增长率 | % | 19.4% |
| 增长率 (g₃) | 10-20 年增长率 | % | 9.7% |
| 永续增长率 (g∞) | Gordon 永续增长率 | % | 2.5% |

### 财务参数

| 参数 | 说明 | 单位 | 示例 |
|------|------|------|------|
| 现金 (X) | 企业持有的现金及等价物 | 元/亿 | 62500 亿 |
| 总负债 (F) | 企业总负债 | 元/亿 | 49500 亿 |
| 股票总数 (N) | 发行在外的股票数量 | 股/亿股 | 23300 亿股 |

## 估值结果

### 三个时间维度

工具同时计算三种不同时间维度的 DCF 估值：

| 时间维度 | 每股价值 | 股权价值 | 显式期现值 | 终值现值 |
|----------|----------|----------|------------|----------|
| 5年 DCF | 基于 5 年现金流 + 终值 | EV + 现金 - 负债 | 1-5 年折现现金流之和 | Gordon 终值折现 |
| 10年 DCF | 基于 10 年现金流 + 终值 | 同上 | 1-10 年折现现金流之和 | Gordon 终值折现 |
| 20年 DCF | 基于 20 年现金流 + 终值 | 同上 | 1-20 年折现现金流之和 | Gordon 终值折现 |

### 估值判断

- **低估**：内在价值高于当前价格 20% 以上
- **高估**：内在价值低于当前价格 20% 以上
- **合理**：内在价值在当前价格 ±20% 范围内

最终判断基于 5年、10年、20年 每股价值的平均值。

## 现金流预测

### 分段增长率

| 年份范围 | 增长率 | 说明 |
|----------|--------|------|
| 1-5 年 | g (38.8%) | 高成长期 |
| 6-10 年 | g₂ (19.4%) | 增速放缓期 |
| 11-20 年 | g₃ (9.7%) | 成熟期 |

### 表格字段

| 字段 | 说明 |
|------|------|
| 年份 | 第 N 年 |
| 增长率 | 对应阶段的增长率 |
| 预测现金流 | FCF × (1 + g)^n |
| 折现系数 | 1 / (1 + r)^n |
| 折现现金流 | 预测现金流 × 折现系数 |
| 累计 | 折现现金流累计值 |

## 计算公式

### 现金流预测

```
1-5年:   FCF × (1 + g)^n
6-10年: FCF × (1 + g)^5 × (1 + g₂)^(n-5)
11-20年: FCF × (1 + g)^5 × (1 + g₂)^5 × (1 + g₃)^(n-10)
```

### 折现计算

```
折现系数 = 1 / (1 + r)^n
折现现金流 = 现金流 × 折现系数
```

### Gordon 终值

```
TV_n = FCF_(n+1) / (r - g∞)
FCF_(n+1) = 第 n 年 FCF × (1 + g∞)
终值现值 = TV_n / (1 + r)^n
```

> 注意：须满足 r > g∞

### 股权价值

```
EV = 显式期折现现金流之和 + 终值现值
股权价值 = EV + 现金 - 负债
每股价值 = 股权价值 / 股票总数
```

## 功能特性

### 保存记录

点击「💾 保存」按钮，可将当前估值结果保存到本地存储：
- 标的名称
- 日期
- 当前价格
- 5年/10年/20年 DCF 每股价值

### 查看历史

点击「📋 记录」按钮，可查看所有保存的估值记录，并支持删除操作。

### 数据持久化

所有记录存储在浏览器 `localStorage` 中，键名为 `dcf_records`。

## 使用示例

以拼多多为例：

```
当前价格: 176 元
自由现金流: 96600 亿
折现率: 17%
增长率: 38.8% → 19.4% → 9.7%
永续增长率: 2.5%
现金: 62500 亿
负债: 49500 亿
股票数: 23300 亿股
```

点击「🚀 执行计算」后，页面会自动计算并展示：
- 三个时间维度的每股价值
- 股权价值
- 估值判断（低估/高估/合理）
