# DCF 永续期估值计算器

白马股 DCF 永续期（Gordon Growth Model）内在价值推导工具，适用于可口可乐、沃尔玛、长江电力等成熟稳定型企业的估值分析。

---

## 一、页面入口

```
白马股 DCF 估值 → DcfMaturity.vue
```

左侧为参数输入区，右侧为估值推导报告展示区。

---

## 二、输入参数说明

### 2.1 核心利润参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **Revenue** | 1000 亿 | 过去 12 个月（TTM）或最新财年的总营业收入。单位：亿美元 |
| **Operating Margin** | 28.7% | 核心营业利润率（EBIT Margin），剔除非经常性损益 |
| **Tax Rate** | 17.8% | 有效税率，即实际缴纳的税率，非单纯法定税率 |

### 2.2 估值驱动因素

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **ROC** | 15% | 资本回报率（Return on Capital）。成熟公司永续期 ROC 通常向 WACC 靠拢 |
| **WACC** | 5.6% | 加权平均资本成本，通常取值 5%~9%。该参数极度敏感，0.5% 的误差将导致估值巨大变化 |
| **g（永续增长率）** | 3.5% | 绝不可超过宏观名义 GDP 增速或 10 年期无风险国债收益率（2%~4%） |

### 2.3 资本结构

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **Cash** | 200 亿 | 现金、现金等价物及短期投资总额。单位：亿美元 |
| **Total Debt** | 300 亿 | 含短债、长债及资本化的经营性租赁总额 |
| **Diluted Shares** | 10 亿股 | 含潜在期权行权后的总股本，防止估值虚高 |

## 三、计算公式推导

### 3.1 Step 1 — 计算基期利润

```
EBIT = Revenue × Operating Margin

NOPAT = EBIT × (1 - Tax Rate)
```

- **EBIT**：息税前利润，反映核心业务盈利能力
- **NOPAT**：税后净营业利润，剔除了税收影响，更真实地反映主业创造价值的能力

### 3.2 Step 2 — 再投资效率

```
再投资率 = g / ROC
```

公司为维持 g% 的永续增长，需要将 NOPAT 的一定比例再投入资本。

**举例**：若 g = 3.5%，ROC = 15%，则再投资率 = 23.3%，意味着每赚 1 元 NOPAT，需再投入 0.233 元以支撑增长。

### 3.3 Step 3 — 预期自由现金流

```
NOPAT(下一年) = NOPAT × (1 + g)

FCFF₁ = NOPAT(下一年) × (1 - 再投资率)
```

- **FCFF**（Free Cash Flow to Firm）：企业自由现金流，即支付所有运营成本、税费和资本支出后，可分配给债权人和股东的自由现金

### 3.4 Step 4 — 核心业务价值

```
核心业务价值 = FCFF₁ / (WACC - g)
```

此为 **Gordon Growth Model（戈登增长模型）** 的核心公式：

$$P_0 = \frac{D_1}{r - g}$$

其中：
- $P_0$：当前股价/价值
- $D_1$：下一年现金流
- $r$：折现率（WACC）
- $g$：永续增长率

**约束条件**：WACC 必须严格大于 g，否则模型失效（分母为负或零）。

### 3.5 Step 5 — 每股价值

```
股权总价值 = 核心业务价值 + Cash - Total Debt

每股价值 = 股权总价值 / Diluted Shares
```

加回现金、减去债务，得到归属股东的真实权益价值，再除以总股本得到每股内在价值。

## 四、使用方法

### 4.1 操作流程

1. **输入参数**：根据目标公司财务数据，填写左侧表单
2. **执行计算**：点击「执行 DCF 估值推导」按钮
3. **查看报告**：右侧自动生成完整的 5 步推导报告
4. **保存记录**：点击「💾 保存」可将本次估值记录存入浏览器本地存储
5. **查看历史**：点击「📋 记录」可查看历史估值记录，支持加载和删除

### 4.2 参数获取渠道

| 参数 | 推荐数据源 |
|------|-----------|
| Revenue | 公司季报/年报，Wind/Bloomberg |
| Operating Margin | 计算得出：EBIT / Revenue |
| Tax Rate | 实际税率 = 所得税费用 / 税前利润 |
| ROC | 通常取 10%~20%，成熟消费品公司偏高 |
| WACC | 可参考十年期国债收益率 + 股票风险溢价 |
| g | 通常取 2%~4%，参考名义 GDP 增速 |
| Cash/Debt | 资产负债表 |
| Diluted Shares | 年报附注中的稀释股数 |

## 五、敏感性分析

### 5.1 关键敏感参数

DCF 估值对以下两个参数极度敏感：

| 参数 | 影响方向 | 建议 |
|------|----------|------|
| **WACC** | WACC ↑ → 估值 ↓ | 使用保守的折现率，可参考 CAPM 模型计算 |
| **ROC** | ROC ↑ → 再投资率 ↓ → FCFF ↑ → 估值 ↑ | 关注公司护城河和竞争优势的可持续性 |

### 5.2 永续增长率约束

```
g 必须 < WACC
g 建议 < 10 年期无风险国债收益率（约 2%~4%）
```

若 g 接近或超过 WACC，模型将失效，页面会弹出错误提示。

---

## 六、局限性

1. **单阶段假设**：永续增长率固定不变，未考虑公司生命周期的阶段性变化
2. **参数主观性**：ROC、WACC、g 均依赖分析师主观判断
3. **忽略增长期权**：初创期或转型期公司的未来增长机会未被纳入
4. **不适合周期股**：强周期行业（钢铁、煤炭、航空）的现金流波动大，DCF 结果参考价值有限

---

## 七、适用场景

- 成熟稳定的消费品企业（可口可乐、茅台）
- 公用事业/基础设施（长江电力）
- 低增速市场的龙头企业（沃尔玛、宝洁）
- 具备强护城河和稳定现金流的"现金牛"公司
